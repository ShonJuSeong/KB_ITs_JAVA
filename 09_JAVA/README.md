# 9. 제너릭
## ✅ 1. 제네릭 (Generic)
📌 제네릭이란?  
클래스나 메서드를 정의할 때 자료형을 미리 지정하지 않고,  
사용할 때 자료형을 지정해서 쓰는 문법.  

🔸 왜 필요할까?  
타입을 명확하게 하여 컴파일 오류 방지  
여러 자료형에 대해 재사용성 높임  
형변환 없이 안전하게 데이터 사용 가능  

## ✅ 2. 예시  
🔸 예제: 제네릭 사용  
ArrayList<String> list = new ArrayList<>();  
list.add("Hello");  
// list.add(123);  ❌ 오류: String만 저장 가능  
String s = list.get(0);  // 형변환 필요 없음!
